# Retto Roulette

네이버 카페 댓글 참여자를 빠르게 불러와 스트리머의 선물 추첨을 진행하는 웹 룰렛입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 품질 확인

```bash
npm test
npm run build
```

## 배포

`docs/`에는 검증된 정적 배포본이 들어 있습니다. GitHub Pages의 배포 원본을
`main` 브랜치의 `/docs` 폴더로 지정하면 바로 공개할 수 있습니다.
Vite의 기본 경로는 `/rettoroulette/`로 설정되어 있습니다.
