# Retto Roulette

스트리머가 카페 댓글 참여자를 빠르게 정리하고, 바로 라이브 추첨을 진행하는 정적 웹 앱입니다.

## 0.2.0 사용 흐름

1. 네이버 카페 게시글을 연 뒤 `Ctrl+A`, `Ctrl+C`로 화면 내용을 복사합니다.
2. Retto Roulette의 붙여넣기 창에 넣으면 브라우저 안에서 댓글 작성자만 파싱합니다.
3. 파싱 결과를 확인하고 참가자 명단에서 추가·삭제·수정합니다.
4. 룰렛 또는 마블 레이스로 라이브 추첨을 진행합니다.

직접 이름을 입력한 경우에도 바로 명단 편집 단계로 들어갑니다. 추첨 전에는 언제나 최종 참가자와 상품을 확인할 수 있습니다.

## 개인정보와 동작 방식

- 복사한 카페 내용은 사용자의 브라우저 안에서만 처리됩니다.
- URL 수집, 네이버 로그인 세션 연결, Cloudflare 프록시/크롤링 서버를 사용하지 않습니다.
- 원문 HTML이나 카페 로그인 정보는 저장하거나 전송하지 않습니다.

## 0.2.0 변경 사항

이 버전은 기존의 URL 기반 댓글 가져오기와 세션·서버 연동 방식을 제거한 **붙여넣기 기반 로컬 파싱**으로 바뀌었습니다. 기존 URL 가져오기 흐름을 사용하던 경우에는 카페 글을 열어 화면 내용을 복사해 붙여넣어 주세요.

## GitHub Pages 배포

이 앱은 별도 백엔드가 필요 없는 정적 사이트이므로 GitHub Pages에 적합합니다. 빌드 결과물은 `docs/`에 두고, GitHub Pages의 배포 원본을 `main` 브랜치의 `/docs` 폴더로 지정하면 됩니다.

Vite 기본 경로는 `/rettoroulette/`입니다.
