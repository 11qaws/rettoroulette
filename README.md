# Retto Roulette

스트리머가 카페 댓글 참여자를 빠르게 정리하고, 바로 라이브 추첨을 진행하는 정적 웹 앱입니다.

## 0.3.0 사용 흐름

1. 네이버 카페 게시글을 연 뒤 `Ctrl+A`, `Ctrl+C`로 화면 내용을 복사합니다.
2. Retto Roulette의 붙여넣기 창에 넣으면 브라우저 안에서 댓글 작성자 명단을 만듭니다.
3. 명단이 맞으면 바로 방송 준비 화면으로 넘어갑니다. 빠진 사람·오타·순서만 필요할 때 명단을 다듬습니다.
4. 사람/상품, 당첨 인원, 후보 수, 룰렛/마블을 정하면서 실제 방송 화면을 미리 봅니다.
5. `이 설정으로 방송 시작`을 누르면 설정은 접히고 룰렛 또는 마블만 크게 남습니다.
6. 한 번에 여러 명을 뽑으면 이번 회차의 당첨자가 번호와 이름으로 모두 남습니다. 다음 추첨을 시작할 때 새 목록으로 바뀝니다.

직접 이름을 입력한 경우에도 바로 명단 편집 단계로 들어갑니다. 방송 중 명단·상품·기록·고급 설정은 `진행 도구`에만 넣어 화면을 정리합니다.

## 개인정보와 동작 방식

- 복사한 카페 내용은 사용자의 브라우저 안에서만 처리됩니다.
- URL 수집, 네이버 로그인 세션 연결, Cloudflare 프록시/크롤링 서버를 사용하지 않습니다.
- 원문 HTML이나 카페 로그인 정보는 저장하거나 전송하지 않습니다.

## 0.3.0 변경 사항

- 방송 전에는 명단·규칙·실제 룰렛 미리보기를 한 화면에서 확인합니다.
- 방송 시작 뒤에는 설정을 접고, 읽기 전용 규칙 한 줄과 `진행 도구`만 남깁니다.
- 다인 추첨 결과는 마지막 한 명만 보여주지 않고, 이번 회차 전원을 순서대로 고정 표시합니다.
- 다음 회차 설정을 바꿔도 현재 결과는 남으며, 실제 다음 추첨을 시작할 때만 새 회차로 교체됩니다.

기존의 붙여넣기 기반 명단 만들기와 브라우저 내 당첨 기록은 그대로 유지됩니다.

## GitHub Pages 배포

이 앱은 별도 백엔드가 필요 없는 정적 사이트이므로 GitHub Pages에 적합합니다. 빌드 결과물은 `docs/`에 두고, GitHub Pages의 배포 원본을 `main` 브랜치의 `/docs` 폴더로 지정하면 됩니다.

Vite 기본 경로는 `/rettoroulette/`입니다.
